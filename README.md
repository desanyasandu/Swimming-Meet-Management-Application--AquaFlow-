# 🌊 AquaFlow | Swimming Meet Management System

A high-performance, single-file web application designed for professional swimming meet management. AquaFlow simplifies the complexity of meet organization, from swimmer registration to automated seeding and results calculation.

![AquaFlow Logo](https://img.shields.io/badge/AquaFlow-Meet%20Manager-blue?style=for-the-badge&logo=water)

## ✨ Key Features

- **🏆 Professional Dashboard**: Real-time statistics of registered swimmers, schools, and active events.
- **📊 Excel Integration**: Bulk import/export capabilities using SheetJS, allowing seamless management via spreadsheets.
- **⚡ Automated Seeding**: Intelligent HY-TEK style lane seeding (center-out) for timed finals.
- **🏫 School & Club Registry**: Manage school zones and institutional registrations easily.
- **🏊 Swimmer Management**: Automated age group detection and entry validation.
- **⏱️ Result Entry & Standings**: Fast result input with automatic point calculation and school rankings.
- **🖨️ Professional Reports**: Print-ready HY-TEK style reports for heats and final results.
- **💾 Local Persistence**: Uses **IndexedDB** for robust, offline-first data storage within the browser.

## 🚀 Getting Started

Simply open `index.html` in any modern web browser. No server setup or installation is required.

### Data Management
1. Go to **Main Settings**.
2. Download the **Master Excel** template.
3. Fill in your data and **Upload Enrollment Sheet** to bulk-import swimmers and schools.

## 🛠️ Technical Stack

- **Frontend**: Vanilla HTML5, CSS3 (Modern Glassmorphism Design)
- **Icons**: [Ionicons](https://ionicons.com/)
- **Typography**: [Google Fonts (Outfit)](https://fonts.google.com/specimen/Outfit)
- **Data Engine**: [SheetJS (xlsx)](https://sheetjs.com/)
- **Storage**: Browser IndexedDB API
